# A simple NodeJS Webapplication that allows yout to Post Tweets

### Getting Started

1. Clone the repo
   ```sh
   git clone https://github.com/Tonycsbe/LB03_M183_AU.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the app
   ```sh
   npm run start
   ```
4. Open the app in your browser
   ```sh
   http://localhost:3000
   ```
